# PortLogger plugin

Plugin for logging messages via CSpect in printf like format. 

Supported flags:
| Flag | Meaning |
| --- | ---- |
| %c  | char |
| %d | integer |
| %i | integer |
| %u | unsigned integer |
| %x | hexadecimal unsigned integer |
| %X | hexadecimal unsigned integer (upper case) |
| %o | octal unsigned integer |
| %f | float |
| %F | float |
| %s | c string |
